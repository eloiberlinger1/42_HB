# 42 School Projects

## Projects Overview

### Core Projects

- **[libft](libft/)** - Custom C library implementing standard functions (string manipulation, memory management, linked lists, etc.)

- **[get_next_line](get_next_line/)** - Function to read a line from a file descriptor, handling multiple file descriptors and variable buffer sizes

- **[ft_printf](ft_printf/)** - Custom implementation of the printf function, supporting various format specifiers (c, s, p, d, i, u, x, X, %)

- **[push_swap](push_swap/)** - Sorting algorithm project using two stacks (A and B) with limited operations, optimizing the number of moves

- **[born2beroot](born2beroot/)** - Linux server configuration project, setting up a Debian-based virtual machine with specific security and monitoring requirements

- **[python_modules](python_modules/)** - Python programming modules and exercises from the new core curicullum of 42 programm.
